## GitHub Pages 設定

| 項目 | 設定値 |
|------|--------|
| ソースブランチ | `main` |
| 公開フォルダ | `/docs` |
| カスタムドメイン | なし（デフォルトドメイン使用） |
| HTTPS | 強制有効 |
| リポジトリ公開設定 | public |

## App Store 申請での使用

App Store Connect でアプリを申請する際、以下の欄にプライバシーポリシーURLを入力する：

- **App Information > Privacy Policy URL:**
  `https://frisk1995.github.io/kashikari-me-beta/privacy-policy.html`

## 注意事項

- GitHub Pages のビルドには数分かかる場合がある。申請前にブラウザでアクセスし、正常に表示されることを確認すること。
- プライバシーポリシーの内容を変更した場合は、App Store Connect のポリシーURLを更新する必要はないが、内容変更日を本ページ末尾に記録すること。
